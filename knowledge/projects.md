# Personal & Featured Projects

## Whale Social Network (Personal Project)
- **Period:** December 2023 – Present
- **Status:** In progress
- **Description:** Personal project inspired by livestream and social network apps. Full-stack implementation exploring modern architecture patterns.
- **Repo (Client):** https://github.com/nbhung278/Whale-SN-Client
- **Repo (Server):** https://github.com/nbhung278/Whale-SN-Server
- **Team size:** Solo project (1 person)
- **Technologies:** NestJS, Next.js, GraphQL, Prisma, PostgreSQL, Zustand, TypeScript, SocketIO, Shadcn, Docker

### What was built:
- Backend: NestJS with GraphQL API, Prisma ORM, PostgreSQL database, Docker containerization
- Frontend: Next.js with TypeScript, Shadcn UI components, Zustand for state management
- Real-time features via SocketIO and GraphQL subscriptions
- Full livestream and social networking functionality

### Why this project?
Built it to deepen knowledge in GraphQL, NestJS, and modern full-stack architecture outside of daily work. Also serves as a playground for experimenting with new tools before applying them in production.
